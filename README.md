## Cloud-back
Бэкэнд приложение сервиса для хранения файлов в облаке.

## Установка зависимостей

```bash
$ npm install
```

## Запуск проекта

```bash
# dev mode
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Приложение запускается на 8080 порте.

## Swagger
Swagger с описанием api проекта можно найти по URL /swagger

## Автотесты
```bash
# юнит-тесты
$ npm run test
# е2е тесты
$ npm run test:e2e
# анализ покрытия
$ npm run test:cov
```
